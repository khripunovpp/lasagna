# 🧺 Склад (страница продуктов)

Раздел **Склад** (заголовок *«Склад»* в приложении) — это место, где хранятся все ваши ингредиенты и материалы, используемые в рецептах.
Каждая сущность здесь — это **продукт**, у которого есть цена, единица измерения, категория, бренд, источник и история изменений.

## 🗃️ Структура страницы

- В заголовке отображается *«Склад»* и количество продуктов рядом (например, *«найдено: 42»*).
- Продукты сгруппированы на странице (по умолчанию — по категориям), способ группировки и порядок сортировки можно поменять.
- На карточке каждого продукта видно:
  - Название продукта (клик — открывает на редактирование).
  - **Цена за единицу** справа (например, `0.018 €/г`). Для очень маленьких значений вместо числа показывается *«меньше <0,01»*.
  - Подпись *«изменён <сколько-то времени назад>»* внизу.

## 📝 Черновики

Если вы редактировали продукт, но не сохранили — наверху появится **баннер черновиков** с уведомлением о несохранённых изменениях. Нажмите на него, чтобы перейти к их просмотру или сохранению.

## ✅ Выделение и массовые действия

Клик по карточке продукта включает **режим выделения**:
- Можно выбрать несколько продуктов сразу.
- Появляется панель с быстрыми действиями: выбрать всё, скрыть выбранное или снять выделение.
- После выбора доступно **удаление** отмеченных продуктов. Кнопка **Экспорт** наверху экспортирует только выбранные позиции (если выделение пустое — все).

## 🔁 Импорт и экспорт

В панели управления вверху страницы:

- ➕ **Добавить продукт** — открывает форму нового продукта.
- **Экспорт** — скачивает продукты в файл (только выбранные, если включён режим выделения, иначе — все).
- **Импортировать** — загружает ранее экспортированный файл. Существующие продукты обновляются при совпадении ID, новые — добавляются.

## 📭 Пустое состояние

Если продуктов ещё нет, на странице отображается *«Продукты не найдены»* и кнопка **«Добавить первый продукт»**.
