# 📖 Рецепты

Рецепт — это основная сущность в системе. Он состоит из одного или нескольких продуктов или подрецептов, их количества, единиц измерения и опционального описания.
С помощью рецепта можно точно рассчитать себестоимость, описать состав и использовать его внутри других рецептов.

## 🧭 Страница «Рецепты»

В заголовке отображается *«Рецепты»* и количество рядом (например, *«найдено: 28»*).

### 🛠️ Верхняя панель

- ➕ **Добавить рецепт** — открывает форму создания нового рецепта.
- **Экспорт** — сохраняет рецепты в файл (только выбранные, если включён режим выделения, иначе — все).
- **Импортировать** — загружает ранее экспортированный файл рецептов.

### 🗂️ Группировка, сортировка и фильтры

Над списком — ряд элементов управления:

- **Фильтры** — сужают список по категории, тегу или тексту.
- **Группировка и сортировка** — выбор способа группировки рецептов (например, по категории, по тегу, по дате создания, по первой букве) и направления (от A к Я или от Я к A).

Внутри каждой группы рецепты сортируются по алфавиту.

### 📁 Папки

Рецепты также можно раскладывать по собственным **папкам**. Рядом с фильтрами есть две иконки-переключателя режима отображения:

- **Группировки** — обычный вид, описанный выше.
- **Папки** — иерархический вид. На странице показываются плитки папок, заход внутрь — по клику. Текущий путь виден в виде «хлебных крошек», начиная с *«Рецепты»*; клик по любому сегменту возвращает на уровень выше.

В режиме папок доступны действия:

- Создать **Новую папку**, указав название, иконку и цвет.
- В меню (троеточие) на плитке папки — **Изменить** или **Удалить** её. При удалении непустой папки её содержимое удаляется вместе с ней.
- Перенести рецепты в папку через действие **Переместить в...** — сначала выделите рецепты (см. [«Выделение и массовые действия»](#-выделение-и-массовые-действия) ниже), затем выберите папку-получатель или *«Корень (без папки)»*, чтобы вынести их наверх.

Выбранный режим отображения запоминается между сессиями.

### 🃏 Карточка рецепта

На карточке каждого рецепта видно:
- **Название** — клик открывает рецепт на редактирование.
- Кнопка **Рассчитать** — открывает страницу анализа стоимости этого рецепта.
- Подпись *«изменён <сколько-то времени назад>»* внизу.

## ✏️ Черновики

Если рецепт был отредактирован, но не сохранён, наверху появится баннер с несохранёнными изменениями. По клику можно перейти к их просмотру или сохранению.

## ✅ Выделение и массовые действия

Клик по карточке рецепта включает **режим выделения**. Появляется панель с быстрыми действиями: выбрать всё, скрыть выбранное или снять выделение. С выбранными элементами можно:

- **Удалить** выбранные рецепты
- **Экспортировать** только выбранные (кнопка «Экспорт» наверху учитывает выделение)

## 📭 Пустое состояние

Если рецептов ещё нет, на странице показывается *«Рецепты не найдены»* и кнопка **«Добавить первый рецепт»**.
