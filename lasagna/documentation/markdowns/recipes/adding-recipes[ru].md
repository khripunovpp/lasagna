# ✏️ Создание и редактирование рецепта

Рецепт — это комбинация ингредиентов (продуктов или подрецептов) с указанием их количества, названия и опционального описания.
Эта форма используется как для *«Добавить новый рецепт»*, так и для экрана редактирования.

## 🧾 Основные поля

### 📛 Название рецепта

Верхнее поле — **название рецепта** (например, *«Карамельный Наполеон»*).

### 📝 Описание

**Текстовое поле с форматированием** — *«Опишите, как приготовить этот рецепт, советы и важные замечания»*. Подходит для технологии, размеров формы, нюансов порционирования и любых других деталей.

### 🥧 Сумма результата (опционально)

Поле с подсказкой *«Сколько порций получится (можно оставить пустым)»* — число, обозначающее **ожидаемое количество порций** рецепта.

- Если поле **пустое**, цена за единицу рассчитывается от суммарного веса ингредиентов.
- Если **заполнено**, цена за единицу считается от этого количества порций — удобно, когда стоимость нужна за порцию, а не за грамм.

## 🧩 Ингредиенты

Раздел **Ингредиенты** перечисляет компоненты рецепта. Для каждой строки можно выбрать один из вариантов:

- **Со склада** — *«Продукт со склада»*: подобрать уже существующий продукт.
- **Как рецепт** — *«Найдите ваш рецепт»*: выбрать другой рецепт как подрецепт.
- **Новый продукт** — *«Или введите название нового продукта которого нет на складе»*: введите название — на складе автоматически создастся новый продукт с выбранной для этой строки единицей измерения. Цена при этом пустая — откройте продукт позже на складе и проставьте цену, чтобы расчёты стали корректными.

Для каждого ингредиента дополнительно указывается:
- **Количество** (в граммах или штуках — *«В грамм»* / *«В шт»*), рядом — переключатель единицы.
- Красный **×** удаляет строку.

Кнопка **«Добавить ингредиент»** добавляет новую строку.

### ➕ Математические выражения

Поле «Количество» принимает **математические выражения**:

- `100+20` → 120
- `400*2` → 800
- `200*0.75` → 150
- `100 + 10%` → 110

Удобно для масштабирования, перевода порций или быстрых вычислений.

## 🔁 Переключатель «компонент / основной»

Под списком ингредиентов есть переключатель с двумя положениями:

- *«Это компонент рецепта»* — рецепт помечается как заготовка, которую планируется использовать внутри других рецептов.
- *«Это основной рецепт»* — самостоятельное законченное блюдо.

Помогает обозначить, как именно используется рецепт.

## 🏷️ Теги и категория

- **Теги** — произвольные ключевые слова для поиска и фильтрации рецептов (например, `веган`, `сезонное`, `декор`, `основа`).
- **Категория** — мульти-выбор: рецепт может относиться сразу к нескольким категориям. Под полем — чипы с самыми часто используемыми категориями для быстрого назначения. Сами категории настраиваются в *«Настройках»*.

## 💾 Сохранение

- **Создать рецепт** / **Сохранить изменения** — сохраняет рецепт.
- После создания появляется небольшой баннер *«Новый рецепт добавлен! Посмотреть?»* со ссылкой на только что созданный рецепт.
- Если покинуть страницу без сохранения, изменения остаются как **черновик** (рядом с заголовком появится индикатор *«сохранено как черновик»*).
- **Удалить этот рецепт** / **Удалить этот черновик** — удаляет рецепт или его текущий черновик.

## 🔗 Дополнительные действия для сохранённого рецепта

Когда вы открываете уже сохранённый рецепт, рядом с заголовком появляется дополнительный ряд кнопок:

- **Рассчитать** — открывает страницу анализа стоимости этого рецепта.
- **Копировать** — создаёт независимую копию рецепта в виде черновика, который можно редактировать отдельно.
- **Поделиться ссылкой** — генерирует ссылку на рецепт. После клика кнопка превращается в **«Ссылка готова, скопировать»** — повторный клик кладёт ссылку в буфер обмена. Тот, кто откроет эту ссылку, увидит рецепт, заранее заполненный в форме *«Добавить новый рецепт»*, с кнопкой **«Сохранить полученный рецепт»**, чтобы добавить его себе на склад.
- **Экспорт** — экспортирует **только этот один рецепт** в файл (отдельно от массового экспорта на странице списка).

> 🔒 Под рядом действий на странице редактирования также отображается время последнего изменения.

---

Редактор рецепта позволяет точно собрать структуру блюда, задать выход и получать достоверную себестоимость — для продажи, планирования или инвойсов.
