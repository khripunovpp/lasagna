# 🏠 Главная страница

После входа в приложение вы попадаете на **главную страницу** (заголовок *«Добро пожаловать»*) — это ваш стартовый экран и точка доступа ко всем основным функциям.

## Что здесь есть:

- 📣 **Промо-виджеты** — короткие анонсы от команды (например, новые фичи или Telegram Mini App). Появляются вверху и могут быть закрыты.
- 🚀 **Онбординг** — короткий чек-лист, который показывается, пока вы не выполнили первые базовые шаги.
- 🕘 **Последние изменения** — карточка *«Последние изменения»* с двумя блоками:
  - **Рецепты** — недавно отредактированные рецепты (название + время с момента изменения).
  - **Продукты** — то же для продуктов.
  - Клик по элементу — открывает его для редактирования. Блок не отображается, пока у вас нет записей.

## Верхняя панель:

Хедер закреплён в верхней части экрана и состоит из секций (слева направо):

- ⬅️ **Назад** — появляется, только если есть куда возвращаться.
- 🥬 **Логотип** — переход на главную из любой точки приложения. Маленькая точка на логотипе означает, что есть непрочитанные промо-виджеты.
- 🔎 **Полнотекстовый поиск** — поиск по рецептам, продуктам и инвойсам.
- **Основная навигация** (центральная пилюля):
  - **Рецепты** — список рецептов.
  - **Склад** — все продукты.
  - **Счета** — список инвойсов.
- ⚙️ **Настройки** — язык, валюта, категории, налоги, реквизиты, бэкапы.
- 🧩 **Виджеты** — вспомогательные инструменты (сейчас — конвертер желатина).
- ❓ **Документация** — открывает эту встроенную документацию.
